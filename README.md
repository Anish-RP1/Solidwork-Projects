# About

Welcome to my Mechanical Design Portfolio.

This repository contains a growing collection of CAD models and engineering drawings I created using **SolidWorks**. These projects have helped to strengthen my understanding of mechanical design, GD&T, and manufacturing principles.

I aim to continually expand this repository with increasingly complex projects as I develop my engineering skills.
